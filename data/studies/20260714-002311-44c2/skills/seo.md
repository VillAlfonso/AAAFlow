# seo — fern study

Radical minimalism with one power move: the sources live in a PUBLIC doc.

## Description template (evidence: all 5 packs)

```
[sponsor line + link (ad)]          <- only when sponsored, always first
[one line: "This is the (unbelievable) story of X." or a single-sentence
 summary of what was done]
[optional: one related-listening/reading link with a word of endorsement]

Sources:
<public link to the research document with every citation>

Music:
<library name>:
<artist - track>  (one per line, grouped by library)

_____
<channel tagline>
```

Real examples: bc3b = "We mapped and animated the Trump assassination
attempt. Events are still unfolding." + sources doc + music credits.
da58 = ad, "This is the unbelievable story of the Unabomber.", two book
links with endorsements, sources, music.

## Rules

1. **ZERO tags.** All five studied videos ship with an empty tag list. Do
   not generate tags for this channel.
2. **No hashtags, no chapter timestamps in the description, no "subscribe"
   line, no emoji.**
3. **The sources document is public and linked** — transparency is the
   brand. Our pipeline: publish the project's research/sources as a public
   gist or doc and link it; if that is not possible yet, list the sources
   inline under "Sources:".
4. Music credits are itemized by library and artist (our ledger already
   holds this; print it).
5. Sponsor copy is one line + one link + "(ad)". Never woven into the
   story summary.
