# thumbnails — fern study

The thumbnail is a CASE FILE, not an emotion poster. It shows the story's
evidence artifacts with utilitarian labels; the face (when present) is the
subject's real-world likeness treated as a document, not a reaction shot.

## Title-vs-thumb contrast law (REQUIRED analysis, user rule 2026-07-14)

Measured pairs (title -> burned thumb text):
- Mapping the Trump Shooting -> "SHOOTER / TRUMP" (map position labels)
- The Kids Who Hacked The CIA -> "Wanted / WikiLeaks" (document stamps)
- The Hunt for the King of the Dark Web -> "INTERPOL / AlphaBay / Tor /
  Bangkok / WANTED" (dossier tags)
- How an FBI Agent Infiltrated the KKK -> "FBI" (badge label)
- The Hunt for America's Smartest Killer -> "IQ 170 / FBI" (case stat)

Rule: **the title tells the story; the thumb text labels the evidence.**
Thumb text is never a sentence, never a kicker phrase, never repeats title
words (exception: a proper noun that IS the evidence, e.g. TRUMP as a map
label). 1-5 short labels, all caps or stamp-style, each attached to a
visual element (arrow, box, badge) — annotation, not caption.

## Composition rules

1. One coherent SCENE (map, dossier, location still) rather than a collage;
   labels float on it like an investigator's markup.
2. Palette: desaturated documentary tones + ONE signal color for the
   markup (red arrow/box or yellow stamp).
3. Faces: real-photo likeness of the actual subject, small-to-medium,
   neutral or candid; never an AI-drawn face, never open-mouth reaction.
4. Text is typeset (real fonts, stamp/mono/label styles), AI glyphs banned.
5. Variance: rotate artifact types across videos (map with arrows /
   wanted-poster dossier / badge+stat / annotated location shot).
